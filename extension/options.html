Tell the extension what your sanctioned identity provider and approved apps are, so it can tell sanctioned access apart from shadow auth. Optionally connect it to your Snout server to assess discovered apps in one click.

One host per line. Logins federated to these are treated as sanctioned SSO.
One registrable domain per line. These never count as shadow.
Never recorded.
Used by the “Assess in Snout” button to POST to /api/assess.
If > 0, the catalog is pushed to /api/catalog automatically (minimum 5 minutes).
Privacy: capture is local to this browser. The extension records only domains and how you authenticated (SSO / social / local password / OAuth scopes) — never passwords, page contents, or full URLs. Data leaves your machine only when you click Export or Assess.
